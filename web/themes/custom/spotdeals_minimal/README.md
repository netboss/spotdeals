# SpotDeals Minimal

Minimal Olivero subtheme for SpotDeals.

## Purpose

This theme exists to give SpotDeals a stable, controlled front-end structure
without committing to a full branding pass yet.

## Current scope

- Uses Olivero as the base theme.
- Adds light custom CSS.
- Provides a dedicated Twig wrapper for the Deals Search (Solr) view.

## Next recommended step

Enable Twig debug locally and confirm the exact template suggestions in use for:
- Deals Search (Solr) main view
- Exposed form
- Result row markup

Once confirmed, add only the smallest necessary overrides.
